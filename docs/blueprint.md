# **App Name**: CaramelPepper

## Core Features:

- Local-First AI Refactoring: Utilize local, quantized LLMs as a tool to analyze code complexity and generate context-aware, privacy-preserving refactoring suggestions directly on the user's machine.
- Code Complexity Analysis: Perform mathematical analysis of user-provided code files or snippets to identify and highlight areas of high complexity.
- Style Detective & Alignment: An AI tool that scans current code for naming conventions, indentation, and formatting, ensuring generated refactoring suggestions align with the user's unique coding style.
- Refactoring Suggestion Interface: Provide a dedicated user interface to display code complexity reports and present AI-generated refactoring suggestions for review, with options to accept or refine.
- Style Vault Management (Local RAG): Allow users to define and manage a local 'Style Vault' — a retrieval-augmented generation (RAG) system providing persistent context and long-term memory for local LLMs, improving refactoring quality.
- Project Context Integration: Enable seamless loading and scanning of local codebases or individual project files to establish deep project awareness without external cloud services or configurations.

## Style Guidelines:

- A dark theme for focused coding environments, with a rich, deep purple (#6A0DAD) as the primary color, conveying sophistication and technical prowess.
- The background color is a professional, stark black (#000000), offering maximum contrast and a non-distracting canvas.
- A vibrant, clear blue (#4FC3F7) serves as the accent color, used to draw attention to critical elements, warnings, and interactive components, providing strong visual contrast against the dark background.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-savvy, and scientific feel. Body text font: 'Inter' (sans-serif) for clear readability in longer descriptions and technical content.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets and output with optimal legibility.
- Clean, precise line-art icons. Use abstract geometric shapes to represent code structures, complexity metrics, and refactoring actions, maintaining a minimalist and professional aesthetic.
- Utilize a structured, multi-pane layout for clear presentation of code, complexity reports, and refactoring suggestions. Emphasize efficient use of space with defined margins and intuitive navigation tailored for developer tools.
- Subtle, functional animations for user feedback, suchs as when complexity reports load, refactoring suggestions are generated, or actions are applied, enhancing the sense of responsiveness without being distracting.
